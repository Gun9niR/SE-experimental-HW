Introduction

The game is simple. Choose the size and type of the maze, selecet starting and finishing point, and the program will try to solve it 4 ways.

Features

  • Two types of mazes: Use the drop-down menu Type to choose between:
    • A connected maze, meaning a guaranteed solution
    • A randomly generated maze with no such property
  • Customizable maze size: Set the number of Columns & Rows in corresponding text boxes. For the second type of maze, Density determines the proportion of accessible area in the maze
  • Togglable animation: Toggle on/off Animated to display/hide the process of generating the maze. Toggle on/off Display process of solution to display/hide the process of solving the maze.
  • Solve the maze in 4 ways: The four algorithms are listed above the canvas. After generating the maze, select the starting/finishing point by clicking on the first canvas.
  • Clear current solution: Click Clear to remove current solution.
  • Interrupt solving process: Tired of waiting? Click Interrupt to stop the program.
  • Auto test: Click Auto test to test the speed of each algorithm. The program will randomly select a set of starting/finishing points and solve the maze for 100 times, recording the time & total number of iterations for each algorithm.

Known issues

The fewer mazes the program is solving, the quicker it is. Besides that, the four canvases are refreshed only when all four algorithms have completed one iteration. So the time consumed by each algorithm doesn't accurately reflect how much faster/slower they are compared with one another.
