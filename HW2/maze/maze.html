Introduction
The game is simple. Choose the size and type of the maze, selecet starting and finishing point, and the program will try to solve it in 4 ways.

Known issues

  • The fewer mazes the program is solving, the quicker it is. Besides that, the four canvases are freshed only when all four algorithms have completed one iteration. So the time consumed by each algorithm doesn't accurately reflect how much faster/slower they are compared with others.
  • There's a slim chance that "Interrupt" button will not work properly. Just reselect the starting point.
